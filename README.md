# Como usar o sistema?
### 1. Abra a planilha Alpha7.xlsx e preencha os dados.

### 2. As informações preenchidas precisam seguir o padrão definido de exemplo na lina 3 da planilha, cada coluna deve seguir esse padrão.
![Imagem 1](https://ibb.co/G39y3VYL)

### 3. Abra o arquivo GeradorAlpha7.exe, e selecione o arquivo padrão, clicando no botão Procurar Arquivos.
![Imagem 2](https://ibb.co/TBDJw362)

### 4. Após selecionar o arquivo clique no botão Gerar Arquivo Alpha7 (Lembrando que o novo arquivo gerado, será salvo nas mesma pasta onde o Gerador se encontra).
![Imagem 3](https://ibb.co/R49g61WB)
