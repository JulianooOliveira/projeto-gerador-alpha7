# Como usar o sistema?
### 1. Abra a planilha Alpha7.xlsx e preencha os dados.

### 2. As informações preenchidas precisam seguir o padrão definido de exemplo na lina 3 da planilha, cada coluna deve seguir esse padrão.
![Imagem 1](https://i.imgur.com/Xz2T4hp.png)

### 3. Abra o arquivo GeradorAlpha7.exe, e selecione o arquivo padrão, clicando no botão \*_Procurar Arquivos_\*.
![Imagem 2](https://i.imgur.com/Vk6tM2s.png)

### 4. Após selecionar o arquivo clique no botão \**Gerar Arquivo Alpha7*\* (Lembrando que o novo arquivo, será salvo na mesma pasta onde o Gerador se encontra).
![Imagem 3](https://i.imgur.com/6tKJSMj.png)
